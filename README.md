## Vergleichende Kulturanalyse
### Nigeria, Deutschland, Mexiko, die Philippinen und Indien werden in Bezug auf ihre Geschlechtertrollen, Akzeptanz von autoritärer Führung und dem Verständnis von Freiheit untersucht

**Authorin:**  Alison Moldovan-Mauer

Wie Sie dieses Repository verwenden:

1. Dieses Repository klonen 
    ```
    git clone https://github.com/Alisonmm222/interkult
    ```
2. ```requirements.txt``` installieren
3. Daten des World Value Survey 7. Welle (2017-2022) Version 6 herunterladen und in ```/data``` speichern
4. Die Analyse in ```/notebooks``` durchlaufen lassen
    ```
    run analysis.ipynb 
    ```
